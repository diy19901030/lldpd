\subsection{\-Detailed \-Description}
\-Functions for buffering data for network sending or receiving. \-Bufferevents are higher level than evbuffers\-: each has an underlying evbuffer for reading and one for writing, and callbacks that are invoked under certain circumstances.

\-A bufferevent provides input and output buffers that get filled and drained automatically. \-The user of a bufferevent no longer deals directly with the \-I/\-O, but instead is reading from input and writing to output buffers.

\-Once initialized, the bufferevent structure can be used repeatedly with \doxyref{bufferevent\-\_\-enable()}{p.}{bufferevent_8h_a1b9a3412e8c8e122c745b33d46b9643e} and \doxyref{bufferevent\-\_\-disable()}{p.}{bufferevent_8h_aa15582fe250247aa98d4897527215727}.

\-When reading is enabled, the bufferevent will try to read from the file descriptor onto its input buffer, and and call the read callback. \-When writing is enabled, the bufferevent will try to write data onto its file descriptor when writing is enabled, and call the write callback when the output buffer is sufficiently drained.

\-Bufferevents come in several flavors, including\-:


\begin{DoxyDescription}
\item[\-Socket-\/based bufferevents ]\-A bufferevent that reads and writes data onto a network socket. \-Created with \doxyref{bufferevent\-\_\-socket\-\_\-new()}{p.}{bufferevent_8h_a501557193aa56c9f4923583d9d734395}.


\item[\-Paired bufferevents ]\-A pair of bufferevents that send and receive data to one another without touching the network. \-Created with \doxyref{bufferevent\-\_\-pair\-\_\-new()}{p.}{bufferevent_8h_af8932010b0a590ddf9848026e8a99059}.


\item[\-Filtering bufferevents ]\-A bufferevent that transforms data, and sends or receives it over another underlying bufferevent. \-Created with \doxyref{bufferevent\-\_\-filter\-\_\-new()}{p.}{bufferevent_8h_ab6dea1351561f1866b0526fc1207bb69}.


\item[\-S\-S\-L-\/backed bufferevents ]\-A bufferevent that uses the openssl library to send and receive data over an encrypted connection. \-Created with bufferevent\-\_\-openssl\-\_\-socket\-\_\-new() or bufferevent\-\_\-openssl\-\_\-filter\-\_\-new(). 
\end{DoxyDescription}
