Convenience macro to iter over every value of an iterable object.

Parameters:
atomThe atom you want to iterate on.
valueAtom that will be used to contain each value.

This macro behaves as a for loop. Moreover, at the end of each iteration, value is deallocated. Don't use it outside of the loop!

lldpctl_atom_t* lldpctl_atom_create ( lldpctl_atom_t *  atom)

Create a new value for an iterable element.

The value is meant to be appended using lldpctl_atom_set(). Currently, there is no way to delete an element from a list. It is also not advisable to use getters on a newly created object until it is fully initialized. If its internal representation is using a buffer, it may not be initialized until the first set.

Parameters:
atomThe atom we want to create a new element for.
Returns:
The new element.
lldpctl_atom_iter_t* lldpctl_atom_iter ( lldpctl_atom_t *  atom)

Return an iterator over a given atom.

If an atom is iterable (if it is a list, like a list of ports, a list of VLAN, a list of neighbors), it is possible to iterate over it. First use this function to get an iterator then use lldpctl_atom_iter_next() to get the next item and lldpctl_atom_iter_value() to the actuel item.

Parameters:
atomThe atom we want to create an iterator from.
Returns:
The iterator or NULL if an error happened or if the atom is empty (check with lldpctl_last_error()).

As a convenience, if the provided atom is NULL, this function will return NULL and no error will be raised.
