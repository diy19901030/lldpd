00163 /* ifi_flags.
00164 
00165    IFF_* flags.
00166 
00167    The only change is:
00168    IFF_LOOPBACK, IFF_BROADCAST and IFF_POINTOPOINT are
00169    more not changeable by user. They describe link media
00170    characteristics and set by device driver.
00171 
00172    Comments:
00173    - Combination IFF_BROADCAST|IFF_POINTOPOINT is invalid
00174    - If neither of these three flags are set;
00175      the interface is NBMA.
00176 
00177    - IFF_MULTICAST does not mean anything special:
00178    multicasts can be used on all not-NBMA links.
00179    IFF_MULTICAST means that this media uses special encapsulation
00180    for multicast frames. Apparently, all IFF_POINTOPOINT and
00181    IFF_BROADCAST devices are able to use multicasts too.
00182  */
